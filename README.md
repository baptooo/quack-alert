## Quack alert

Transform the alert function in a more awesome version with a Quack sound !

[Installation](#installation)
[Usage](#usage)

### Installation

```
$ npm install --save-dev quack-alert
```

### Usage

1. with html

```html
<script src="path/to/quack-alert.js"></script>
```

2. with browserify

```js
require('quack-alert');
```
